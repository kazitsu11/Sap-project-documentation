# Order-to-Cash (O2C) Workflow Implementation in Dolibarr ERP

## Project Overview

This project demonstrates the complete **Order-to-Cash (O2C)** business process executed in **Dolibarr ERP**. The workflow covers the full cycle from customer creation to final payment receipt.

The objective of this project is to understand how ERP systems automate sales, billing, and payment operations.

---

## O2C Workflow Steps Implemented

1. **Company Setup**

   * Configured ERP environment in Dolibarr Demo.

2. **Customer Master Creation**

   * Created customer record: **ABC Traders**

3. **Product Master Creation**

   * Added product: **Laptop Model X (LAP001)**

4. **Sales Order Creation**

   * Created and validated customer sales order.

5. **Invoice Generation**

   * Generated customer invoice from sales order.

6. **Payment Receipt**

   * Recorded payment via bank transfer.

7. **Order Closure**

   * Invoice status updated to **Paid**.
